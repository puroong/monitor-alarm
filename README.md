# monitoring-alarm
WC 모니터링용 알람

~+컴퓨터로 반복적인 일할 때 도움 될 것 같음~

# Prerequisite
QT5 권장함
(최소 QT4 이상)

```
#Ubuntu16.04
#sudo apt-get install qt5-default

#arch
#pacman -S qt5-base
```

# Build(Only Linux)
```
cd /project/root/directory

chmod +x install.sh
./install.sh
```
